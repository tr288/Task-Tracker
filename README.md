# Task Tracker

A responsive web application that helps users organize and manage their daily tasks efficiently. The application allows users to create an account, log in securely, and maintain a personalized task list using browser Local Storage.

## Features

- User Sign Up and Login
- Add New Tasks
- Delete Tasks
- User-specific Task Storage
- Dark / Light Mode
- Responsive Design
- Data Persistence using Local Storage

## Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage

## Project Structure

```
Task-Tracker/
│── index.html
│── style.css
│── script.js
│── README.md
```

## How to Run

1. Clone the repository.

```
git clone <repository-url>
```

2. Open the project folder.

3. Open `index.html` in any modern web browser.

No additional installation is required.
